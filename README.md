# FizzBuzz問題 解答プログラム

1から20までの数を出力し、3の倍数のときは「Fizz」、5の倍数のときは「Buzz」、両方の倍数（15）のときは「FizzBuzz」を表示するJavaScriptプログラムです。

## 制作および登録のプロセス

### 1. アルゴリズムの設計
* **ループ処理：** 1から20までの数値を順に処理するため、固定回数のループに適した `for` 文を採用しました。
* **条件分岐の優先順位：** 3と5の最小公倍数である「15（両方の倍数）」の判定を最初に行わないと、個別の倍数判定（3または5）に先に引っかかってしまうため、`if (i % 3 === 0 && i % 5 === 0)` を最初の分岐条件として設計しました。

### 2. ローカルでの開発・検証環境
* **使用ツール：** Visual Studio Code
* **実行環境：** Node.js (v18.x)
* **検証：** ターミナル上で `node fizzbuzz.js` を実行し、1から20までの出力結果が仕様通りであることを目視にてテスト・確認しました。

### 3. GitHubへの登録（Git操作プロセス）
以下のコマンドをローカル環境のターミナルで順に実行し、本リポジトリへ登録（Push）を行いました。

1. **リポジトリの初期化：** `git init`
2. **ファイルのステージング：** `git add fizzbuzz.js README.md`
3. **ローカルコミット：** `git commit -m "feat: FizzBuzzプログラムとREADMEの初期追加"`
4. **メインブランチの設定：** `git branch -M main`
5. **リモートリポジトリの紐付け：** `git remote add origin [リポジトリのURL]`
6. **リモートへのプッシュ：** `git push -u origin main`
